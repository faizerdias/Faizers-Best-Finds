PUBLISH FREE WITH GITHUB PAGES
1. Sign in to GitHub as faizerdias.
2. Create a new PUBLIC repository named: faizers-best-finds
3. Upload everything inside this folder, keeping the folders intact.
4. Open Settings > Pages.
5. Under Build and deployment, choose "Deploy from a branch."
6. Select branch "main" and folder "/ (root)", then Save.
7. Your address should become:
   https://faizerdias.github.io/faizers-best-finds/
