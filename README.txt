COMMENTS AND SHARED REACTIONS — ONE-TIME GITHUB SETUP
The pages use Utterances, which stores public comments in GitHub Issues.
1. Go to https://github.com/apps/utterances
2. Install the Utterances app for the repository:
   faizerdias/faizers-best-finds
3. In the repository, create a label named:
   website-comment
4. Visitors will need a GitHub account to comment.
5. Their comments and reactions will appear publicly beneath each review.
